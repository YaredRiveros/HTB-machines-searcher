# HTB-machines-searcher
Proyecto en bash que automatiza la búsqueda de máquinas de HTB, escrapeando la data de la API pública https://htbmachines.github.io/bundle.js. Este proyecto fue realizado como parte del curso Hack4u de S4vitar.

Las consultas que soporta son:
- Sincronización con api en nube
- Búsqueda por nombre
- Búsqueda por IP
- Búsqueda por dificultad
- Búsqueda por sistema operativo
- Búsqueda por skills
- Obtención del link de resolución
- Mostrar panel de ayuda

Para conocer cómo usar el script, utilize ./script -h
